## วัตถุประสงค์ของโปรเจกต์

โปรเจกต์นี้ถูกพัฒนาเพื่อการศึกษาและการฝึกฝนทักษะการเขียนโค้ดในด้าน HTML, CSS, JAVASCRIPT โดยมีวัตถุประสงค์เพื่อเรียนรู้การสร้างเว็บไซต์และการจัดการกับฟังก์ชันต่างๆ ของเว็บไซต์จำลอง

## ขอบเขต

โปรเจกต์นี้เป็นการจำลองหรือเลียนแบบเว็บไซต์จริงเพื่อศึกษาและทดลองใช้งานเทคโนโลยีและแนวทางต่างๆ
